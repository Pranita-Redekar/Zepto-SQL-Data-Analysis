<h1 align="center">🛒 Zepto SQL Analytics Project</h1>

<p align="center">
SQL-based data analysis project to explore pricing, discounts, and inventory insights.
</p>

<hr>

<h2>📌 Project Overview</h2>
<p>
This project focuses on analyzing product data from Zepto using PostgreSQL. 
The objective is to extract meaningful insights related to product pricing, discounts, 
stock availability, and category performance using SQL queries.
</p>

<h2>🗂️ Dataset</h2>
<p>
The dataset contains product-level details such as category, product name, MRP, discount percentage, 
discounted price, stock availability, and weight.
</p>

<p>
<b>📂 Dataset:</b> 
<a href="https://github.com/Pranita-Redekar/Zepto-SQL-Data-Analysis/blob/main/zepto_v2.csv" target="_blank">View Dataset</a>
</p>

<h2>⚙️ Tools & Technologies Used</h2>
<ul>
  <li>PostgreSQL</li>
  <li>SQL</li>
</ul>

<h2>🧹 Data Cleaning</h2>
<ul>
  <li>Checked and handled NULL values</li>
  <li>Removed products with zero price</li>
  <li>Converted price values from paise to rupees</li>
</ul>

<h2>📊 Analysis</h2>
<ul>
  <li>Calculated total number of products</li>
  <li>Identified distinct product categories</li>
  <li>Analyzed in-stock vs out-of-stock products</li>
  <li>Detected duplicate product names</li>
  <li>Filtered products with high discounts</li>
  <li>Sorted products based on highest MRP</li>
  <li>Counted products in each category</li>
  <li>Calculated average discounted price per category</li>
  <li>Identified top 5 products with highest discount amount</li>
  <li>Found products with above-average discounts</li>
  <li>Identified out-of-stock products</li>
  <li>Categorized products based on discount (High, Medium, Low)</li>
  <li>Found category with highest average MRP</li>
  <li>Ranked products by price within each category using window functions</li>
  <li>Identified second highest priced product</li>
  <li>Filtered products with low stock but high price</li>
  <li>Calculated total inventory value</li>
  <li>Calculated total available stock</li>
  <li>Identified least discounted product</li>
  <li>Filtered products based on price conditions</li>
  <li>Compared products with average MRP</li>
  <li>Identified product with highest stock</li>
  <li>Found top categories with highest average discount</li>
  <li>Calculated price per gram for better value comparison</li>
  <li>Categorized products based on weight (Low, Medium, Bulk)</li>
  <li>Calculated total inventory weight per category</li>
</ul>
